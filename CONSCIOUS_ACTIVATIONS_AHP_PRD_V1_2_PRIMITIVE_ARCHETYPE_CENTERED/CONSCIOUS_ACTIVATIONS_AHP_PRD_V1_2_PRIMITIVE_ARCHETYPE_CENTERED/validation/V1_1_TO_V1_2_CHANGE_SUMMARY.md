# PRD V1.1 to V1.2 Doctrine, Primitive, and Archetype Correction

## Why this amendment exists

V1.1 correctly made the interview source package the root production object, but it underrepresented the system that turns expression into Activative content. V1.2 restores the missing production spine:

> **Content activates when it gives the viewer a psychological role inside a tension.**

Matrix of Edging → Primitive Families and Bindings → Primitive Coalition Contract → Coalition Signature → Edge Product → Archetype Coalition → Guest Voice DNA Final Script → Composition Before Editing → category-native embodiments → Primitive and Activative evaluation.

## Product corrections

- Makes Interview-First, Expression-First, Primitive-First, Composition-Before-Editing, Visual Research Before Generation, Real-Life Reference Before Generation, Source Fidelity, Negative Space, and Edge Integrity / Anti-Centroid binding doctrines.
- Makes Primitive Families, Bindings, Coalitions, signatures, Edge Products, misuse risks, and evaluation receipts canonical product objects.
- Requires an archetype coalition and Guest Voice DNA Final Script before every composition.
- Makes Carousels and SuperVisuals text-bearing compositions Voice-DNA rewrites rather than quote-only derivatives.
- Makes a reusable 2D Animation Scene Package mandatory for every approved Final Script; full rendering remains archetype/composition driven.
- Adds Brand Genesis Session, Brand Context Version, Voice DNA, Visual DNA, RSCS, CCV, CBAR, SDA, SFL, and Distillation Layers.
- Removes externally imposed creative-filtering logic from normative product behavior. The operator supplies source authority and intended use; the system preserves provenance and scope.

Historical source snapshots remain byte-preserved as evidence and are not normative product language.
