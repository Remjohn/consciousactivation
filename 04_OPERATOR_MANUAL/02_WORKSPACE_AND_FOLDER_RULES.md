# Workspace and Folder Rules

- Maintain one active folder per repository.
- Keep old whole-folder checkpoints as immutable ZIPs under Program Control, not inside active repositories.
- Store current authorities centrally under `CMF_PROGRAM_CONTROL/01_PRODUCT_AUTHORITIES/`.
- Use `docs/constitutional-alignment/` for non-destructive change overlays.
- Use `PROGRAM_STATUS_EXPORT.yaml` for cross-repository status exchange.
- Shared contracts move only through versioned releases in `02_CROSS_REPO_CONTRACTS/`.
