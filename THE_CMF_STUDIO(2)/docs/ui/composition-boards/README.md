# CMF Studio - 24 Visual Composition Approval Boards

These boards show the visual grammar for the four canonical CMF video slots before JSON template and renderer implementation.

## Runtime Explanation

- `CMF_COMPOSITION_RUNTIME_BEHIND_THE_SCENES_DIAGRAM.png`
- `CMF_COMPOSITION_RUNTIME_BEHIND_THE_SCENES.md`

## Current Primitive-Gated V2 Boards

These are the current reference boards after applying the composition primitive triad rule from:

- `registries/evals/composition/cmf_composition_primitive_triads.v1.json`
- `docs/tech-specs/TS-CMF-079-route-specific-visual-feel-and-primitive-composition-gates.md`

Each board contains six vertical composition directions and must be interpreted through the minimum three-primitives gate:

- meaning transform
- delivery shape
- format/material
