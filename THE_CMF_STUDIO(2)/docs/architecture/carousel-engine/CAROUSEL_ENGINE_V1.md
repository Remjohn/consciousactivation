# Carousel Engine V1

Carousel Engine V1 converts source truth, brand context, primitive intent, visual preproduction, asset intelligence, and style route decisions into a coherent multi-slide social carousel with deterministic slide renders, sequence evaluation, operator revision, and export.

Runtime doctrine:

```text
Pydantic = typed contract authority
DSPy = bounded reasoning/compiler layer
Deterministic Python = gates, state transitions, receipts
Agents/sub-agents = orchestration and skill invocation
Provider adapters = materialization only after preflight
Deterministic renderer = final slide rendering
```

Carousel is a multi-frame argument/story/teaching sequence. It owns sequence strategy, slide roles, claim/source mapping, visual rhythm, slide composition, batch render, sequence eval, revision, and export.
