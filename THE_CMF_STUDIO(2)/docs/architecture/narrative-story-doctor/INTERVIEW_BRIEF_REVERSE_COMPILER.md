# Interview Brief as Forward Compiler

The Interview Brief creates the path that questions should follow.

```text
Brand / Doctrine / Primitive / Archetype / Ingredient Need
→ Interview Brief
→ Question Production Contract
→ Answer Acquisition Target
```

Extraction reverses that path:

```text
Answer / Transcript / Expression Moment
→ Actual Ingredient
→ Archetype / Primitive / Format Packet
→ Coverage + Gap Report
```

## Core rule

```text
The Interview Brief is a prior, not evidence.
```

If the brief expected a memory object but the answer never mentions one, extraction must mark `MISSING_MEMORY_OBJECT_SIGNAL`; it must not invent the object.
