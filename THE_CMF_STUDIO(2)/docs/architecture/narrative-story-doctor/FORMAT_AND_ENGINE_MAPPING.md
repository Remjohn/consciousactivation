# Format and Engine Mapping

## Format 01 — Cinematic Story Commentary

Requires:

```text
A-roll spine
emotional change
memory/proof object
B-roll contrast or foreshadowing
sonic story seed
cut-question chain
```

Emits:

```text
Format01StoryExtractionPacket
```

## Format 02 — 2D Avatar / Paper-cut Animated Explainer

Requires:

```text
teachable mechanism
step sequence
concept nouns
diagram potential
avatar gesture opportunity
```

Emits:

```text
Format02ExplainerExtractionPacket
```

## Format 03 — Living Commentary Reactions

Requires:

```text
proof object or quote surface
coach judgment/reaction angle
annotation-worthy phrase
audience recognition trigger
```

Emits:

```text
Format03ReactionExtractionPacket
```

## Format 04 — Conscious Reactions Editing

Requires:

```text
debate tension
ranking/poll/tier-list structure
score-state potential
meme mechanism
```

Emits:

```text
Format04ConsciousReactionExtractionPacket
```

## SuperVisual

Requires one source truth and one visual hook.

## Carousel

Requires thesis, claim sequence, viewer-state sequence, and closure.

## Meme / Poll / Reaction

Requires compressed contradiction, audience tension, or reaction-worthy seed.
