# Narrative Story Doctor + Content Extraction Intelligence V1

## Definition

Narrative Story Doctor + Content Extraction Intelligence V1 is the shared compiler that transforms raw transcripts, Interview Briefs, and Complete Expression Sessions into source-faithful production packets for SuperVisual, Carousel, Video, Format Intelligence, Meme/Poll assets, and Reaction Seeds.

It is not a summarizer.

It compiles:

```text
source truth
→ clusters
→ meaning candidates
→ archetype programs
→ primitive coalition candidates
→ delivery recipe programs
→ format expression programs
→ engine packets
```

## Core laws

```text
The Interview Brief is the forward compiler.
The Story Doctor is the reverse compiler.
The brief can bias extraction but cannot become evidence.
Archetypes are meaning compilers.
Primitive coalitions are force compilers.
Formats are expression compilers.
Component engines are target backends.
```
