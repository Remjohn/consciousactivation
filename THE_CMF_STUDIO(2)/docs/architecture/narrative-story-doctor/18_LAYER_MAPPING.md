# 18-Layer Mapping

Content Extraction Intelligence is a cross-layer compiler.

| Layer | Extraction role |
|---:|---|
| 1 Doctrine | Enforces interview-first, source fidelity, anti-centroid, real-reference-first rules |
| 2 Brand / Workspace | Loads brand_context_version_id, Voice DNA, Visual DNA, Identity Pack |
| 3 Primitive System | Extracts primitive candidates and coalition signatures |
| 4 Reasoning | Runs CBAR, SDA, SFL, RSCS, Matrix of Edging, T-Code, V-Code where relevant |
| 5 Research | Uses Guest Dossier, Audience Reality, research gaps |
| 6 Interview Intelligence | Owns brief binding, transcript beats, expression moments |
| 7 Content Strategy | Extracts archetype/content asset candidates |
| 8 Sequencing | Emits viewer-state sequence, open loops, closure contracts |
| 9 Creative Ingredients | Extracts proof/memory/B-roll/micro-semiotic ingredients |
| 10 Asset Intelligence | Emits asset queries and requirements |
| 11 Visual Style / Motion Skills | Emits style route hints without owning route execution |
| 12 Visual Preproduction | Emits storyboard ingredients and VCP seeds |
| 13 Composition | Emits frame profile, composition role, attention-path hints |
| 14 Component Engines | Emits packets for SuperVisual, Carousel, Video, etc. |
| 15 Providers / Tools | Emits provider needs only |
| 16 Render / Timeline | Emits source timing, captions, sonic seeds |
| 17 Evaluation / Review | Emits receipts and blockers |
| 18 Publishing / Memory | Emits content asset codes and learning memory seeds |
