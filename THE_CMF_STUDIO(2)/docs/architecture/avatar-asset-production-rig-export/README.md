# Avatar Asset Production + Rig Export V1

## Definition

This layer defines how the avatar physically exists.

It produces manifests and QA receipts for:

```text
face expression plates
paper body layers
PSD/layer requirements
Stretchy Studio import
Spine export
DragonBones export
Remotion avatar layer payload
action timelines
character QA
rig continuity
```

It does not decide performance. Avatar Performance Layer decides performance.
