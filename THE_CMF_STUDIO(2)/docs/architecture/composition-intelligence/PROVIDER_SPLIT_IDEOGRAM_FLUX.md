# Provider Split: Ideogram + Flux

## Ideogram

Role:

```text
composition_plate_generator
```

Allowed:

```text
paper-cut/anime editorial layout
graphic balance
avatar placement
rough typography preview
placeholder zones for real objects
```

Forbidden:

```text
final proof object generation
extra objects
extra text
layout drift
cognitive load violations
```

## Flux

Role:

```text
reference_based_object_editor
```

Allowed:

```text
insert real-life cutout reference
paperize object
harmonize lighting
add contact shadow
preserve locked composition
```

Forbidden:

```text
rewrite text
move avatar
change layout
invent claims
add objects
```
