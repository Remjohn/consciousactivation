# Format 02 Composition Laws

```text
1. One concept per scene.
2. Text is revealed, not displayed all at once.
3. Avatar performs; text anchors.
4. Real-life cutout gets one job.
5. One audience proxy per scene, except final recap.
6. Animation replaces text where possible.
7. One avatar action per scene.
8. Objects move like paper, not AI video.
9. One concept = one primary visual action.
10. Composition locks before provider edit or video timeline.
```
