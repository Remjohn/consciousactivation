# Composition Intelligence Core + Format 02 Pack V1

## Definition

Composition Intelligence V1 is the shared layer that locks where things go, how much can be visible, what the viewer should look at first, what providers are allowed to change, and whether a scene is visually safe to animate.

It answers:

```text
Where does everything go?
How much text is allowed?
What is the attention path?
Which layers exist?
Which elements are locked?
Which provider can generate/edit which parts?
What is too much cognitive load?
```

## Non-redundancy

```text
Format Intelligence:
This should become Format 02 and requires avatar / paper-cut explainer grammar.

Composition Intelligence:
This exact scene can show one concept, one avatar action, one hero object, one proxy, and max 14 visible words.
```
