# Asset Intelligence Studio V1 UI Plan

Asset Intelligence Studio is the future operator surface for assets and ingredients.

It is not required for the first backend patch, but the backend contracts are shaped for it.

## Layout

```text
Top:
Brand / Project / Asset Mode / Filters

Left:
Asset Collections
- Brand Assets
- Guest Assets
- Proof Objects
- Memory Objects
- Micro-Semiotic Anchors
- Acting Library
- Paper-Cut Artifacts
- Style References
- Generated Variants
- Winners
- Blocked / Deprecated

Center:
Asset Grid + Reference Board Canvas

Right:
Inspector
- Identity
- Source / Provenance
- Semantic Profile
- Primitive Fit
- Style Route Fit
- Frame Profile Fit
- Usage History
- Performance
- Fatigue
- Actions

Bottom:
Receipts / Versions / Evals / Usage Timeline
```

## Operator actions

```text
approve asset
mark reference-only
block asset
promote winner
request variant
attach to primitive
attach to style route
attach to Visual Schema
send to SuperVisual
send to Carousel
send to Video
```
