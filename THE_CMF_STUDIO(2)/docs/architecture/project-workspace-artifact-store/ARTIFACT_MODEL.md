# Artifact Model

## ArtifactRef

A pointer to an artifact, not the raw file bytes.

## ArtifactVersion

An immutable version of an artifact.

## ArtifactManifest

A collection of artifact refs for a run or workspace.

## ArtifactLineage

A source-to-derived relationship between artifacts.

## ArtifactReceipt

A verification/evaluation record for an artifact.

## RunArtifactDirectory

The deterministic folder tree for a specific run.
