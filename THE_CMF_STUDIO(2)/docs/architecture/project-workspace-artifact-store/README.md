# Project Workspace + Artifact Store V1

## Definition

Project Workspace + Artifact Store V1 defines deterministic client workspace folders, artifact references, manifests, lineage records, and receipts.

It is the filesystem backbone for:

```text
client context
references
avatar libraries
template atlases
provider outputs
video timelines
local render outputs
export packs
receipts
```

V1 is local filesystem based and does not call providers or renderers.
