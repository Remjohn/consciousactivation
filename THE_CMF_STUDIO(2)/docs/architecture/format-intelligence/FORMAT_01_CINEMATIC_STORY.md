# Format 01 — Cinematic Story Commentary

## Purpose

A-roll-led interview story where B-roll, sonic design, contrast cuts, memory objects, and composition guide the viewer through emotional change.

## Required

```text
A-roll story spine
emotional change map
cut-question chain
memory or proof object
B-roll contrast / foreshadowing function
power phrase plan
sonic story arc seed
```

## Forbid

```text
generic stock B-roll
pretty shot with no story function
synthetic primary delivery
sound overcompensation
text overload
motion outrunning the voice
```

## Allowed style routes

```text
CAC
Documentary Proof
Paper-Cut Artifact for source-grounded derivatives
```
