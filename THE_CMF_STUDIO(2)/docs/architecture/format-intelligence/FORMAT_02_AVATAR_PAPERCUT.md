# Format 02 — 2D Avatar / Paper-cut Animated Explainer

## Purpose

Step-by-step module teaching, conceptual reframing, and mechanism explanation through avatar performance, paper-cut objects, and diagram choreography.

## Required

```text
teachable mechanism
concept nodes
diagram sequence
avatar or paper-cut character policy
concept-motion budget
clean voice mix
```

## Forbid

```text
decorative avatar
overloaded diagram
motion not tied to concept timing
childish paper-cut aesthetic unless intentional
```
