# Extraction Intelligence vs Format Intelligence

Extraction Intelligence asks:

```text
What source-faithful material did we capture?
What archetype does it match?
What primitive coalition does it imply?
Which engine packets could it become?
```

Format Intelligence asks:

```text
Given the extracted packet, what exact format grammar must be used?
What ingredients are mandatory?
What style routes are allowed?
What motion/sound doctrine applies?
What eval gates must pass?
What render requirements must be compiled?
```

Example:

```text
Extraction:
This quote has a memory object, emotional pause, and Relief Peak archetype.

Format Intelligence:
For Format 01, express it as A-roll-led cinematic story with memory-object B-roll, slow drift, voice-first sound, sparse power phrases, and CAC / Documentary Proof style routes.
```
