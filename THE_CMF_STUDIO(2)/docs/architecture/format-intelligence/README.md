# Format Intelligence V1

## Definition

Format Intelligence V1 is the shared realization compiler that converts extraction packets, archetype programs, primitive coalition candidates, delivery recipes, brand context, and audience reality into format-specific production programs.

It answers:

```text
What does this format require?
What does it forbid?
Which ingredients are mandatory?
Which composition grammar applies?
Which layer stack is required?
Which sound/motion doctrine applies?
Which style routes are allowed?
Which eval gates must pass?
Which component engine receives the result?
```

## Non-redundancy with Extraction Intelligence

```text
Extraction Intelligence = source-to-meaning compiler.
Format Intelligence = meaning-to-format-realization compiler.
```

Extraction finds ingredients. Format Intelligence writes the recipe.
