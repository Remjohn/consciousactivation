# Format 04 — Conscious Reactions Editing

## Purpose

High-arousal debate, ranking, tournament, tier-list, and myth-breaking reaction format.

## Required

```text
debate tension
reaction UI surface
ranking / poll / score / tier state
meme mechanism
high-arousal justification
```

## Sound

```text
memetic cue limit = 1 per 10 seconds
```

## Forbid

```text
zoom spam
meme cue trivializing serious source
ranking surface without real tension
score state not tied to argument shift
```
