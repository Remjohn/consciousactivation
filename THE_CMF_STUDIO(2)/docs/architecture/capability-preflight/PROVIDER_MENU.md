# Provider Menu

The Provider Menu summarizes provider readiness before production.

Example roles:

```text
Ideogram = composition_plate_generator
Flux = reference_based_object_editor
OpenAI image = fallback image provider
RunningHub / ComfyUI = optional workflow providers
```

The menu must show:

```text
configured
available
missing secrets
degraded status
estimated cost
sample required
sample approved
batch readiness
```
