# Audience Proxy Personas

The four audience proxies represent viewer states, not demographics.

```text
Confused Seeker
Overwhelmed Doer
Skeptical Protector
Gentle Builder
```

## Roles

```text
Confused Seeker = open question / uncertainty
Overwhelmed Doer = pressure visibility / human affinity
Skeptical Protector = proof demand / discernment
Gentle Builder = expected future value / closure
```
