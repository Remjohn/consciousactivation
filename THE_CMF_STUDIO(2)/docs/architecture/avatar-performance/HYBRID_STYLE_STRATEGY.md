# Hybrid Style Strategy

The recommended character direction is:

```text
storybook / anime-inspired expression plates
+ paper-cut body
+ real-life object cutout environment
+ audience proxy stick figures
```

The face is not video-real. It is an approved expression plate with paper texture, cutout border, consistent shadow, and minor head motion only.

Avoid:

```text
continuous face morphing
lip-sync mouth flaps
uncanny realistic facial deformation
```
