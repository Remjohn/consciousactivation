# Avatar Performance Layer V1

## Definition

Avatar Performance Layer V1 is a reusable, no-lip-sync, limited-performance subsystem for the coach guide avatar and audience proxy personas.

It serves all formats, but is optimized for:

```text
Format 02 — 2D Avatar / Paper-cut Animated Explainer
```

## Core formula

```text
storybook / paper-cut coach avatar
+ expression plates
+ rigged paper body
+ gesture clips
+ prop attachments
+ audience proxy personas
+ deterministic performance plan
```

## Non-goal

This is not a talking avatar system.
