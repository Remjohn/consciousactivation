# Canonical Creative Pipeline Doctrine V1

## One-line doctrine

Every asset begins with human/source truth, is interpreted through primitives, grounded by research, shaped by sequencing, prepared through visual schema and storyboard reasoning, routed through a style skill, composed before editing, rendered deterministically, evaluated through receipts, approved by an operator, and remembered through performance.

## Hard laws

1. Primitive coalition before archetype/style/provider.
2. Visual research before generation for high-value assets.
3. Real-life reference before generation unless explicitly abstract.
4. Composition before editing.
5. No provider job without source reference, style route, frame profile, composition role, and primitive binding.
6. 16:9 is source-only for short-form.
7. Final render is deterministic.
8. Every output creates receipts and memory.
