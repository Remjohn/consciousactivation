## Runtime policy

Pydantic validates all objects.

DSPy may be used for:

```text
research synthesis
recognizable context selection
subjective distortion mapping
PRIMAL interpretation
anti-cliché repair suggestions
visual beat planning
commander rationale
operator feedback parsing
```

Deterministic code must enforce:

```text
brand_context_version_id
source evidence coverage
registry validity
16-element schema completeness
blocking violation handling
packet freeze state
authorization before downstream batch generation
```
