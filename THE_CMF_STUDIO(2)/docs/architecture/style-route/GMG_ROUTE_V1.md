# GMG Route V1

GMG is Generative Motion Graphics. It is expert-routed, not generic motion design.

Core law:

```text
Verbatim Noun Extraction Law
```

Visual nouns must come from source script, VisualSchema, primitive obligations, or approved Asset Intelligence records.

Experts:

```text
01 Neo-Schematic Architect — systems, networks, signal flows, process maps
02 Mono-Kinetic Protagonist — single protagonist, pressure, natural force
03 Emotional Animator — stick figure + photo cutout + paper texture; requires photo cutout
04 Paper Architect — documents, evidence, archives, receipts; requires document/evidence/archive input
05 Editorial Scribe — metrics, frameworks, infographics; forbids 3D/glass/neon
06 Visual Synthesizer — pure geometry, black void, white lines; forbids gold/photo/paper texture/random objects
```
