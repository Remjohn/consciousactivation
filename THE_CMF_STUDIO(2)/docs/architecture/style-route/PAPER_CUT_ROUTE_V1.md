# Paper-Cut Route V1

Paper-Cut splits into two routes.

Paper-Cut Artifact is for proof objects, memory objects, real-life artifacts, micro-semiotic anchors, and source-grounded cutouts. It requires a source object or artifact reference.

Paper-Cut Editorial is for editorial cards, collage-based SuperVisuals, carousel slides, proof cards, and tactile explainers. It requires visual hierarchy, paper surface, layer system, object anchors, type/label treatment, frame profile, and mobile readability.
