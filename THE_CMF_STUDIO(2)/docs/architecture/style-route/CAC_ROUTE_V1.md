# CAC Route V1

CAC means Conscious Ambient Cinema: editorial realism, cinematic B-roll, memory scenes, coach/founder atmosphere, lifestyle inserts, documentary plates, and soft-rounded cinematic cards.

Required principles:

```text
Anchor
Contact
Composition
Atmosphere
Imperfection
Lens
```

Hard gates:

```text
requires real-life/source reference
requires environment logic
requires motivated light
requires mundane anchor or human-scale context
rejects generic cinematic gloss
rejects luxury stock aesthetic unless source-backed
rejects abstract symbolic scenes without real grounding
```
