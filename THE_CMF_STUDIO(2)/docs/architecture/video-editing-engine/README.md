# Video Editing Engine V1

## Definition

Video Editing Engine V1 is the timeline compiler and render-orchestration engine.

It converts:

```text
source media
+ Format Intelligence programs
+ locked CompositionScenePrograms
+ AvatarPerformancePlans
+ source refs
+ asset refs
+ sound doctrine
```

into:

```text
VideoTimelineProgram
RemotionInputProps
OTIOAuditTimeline
ProxyRenderReceipt
FinalRenderReceipt
VideoEvaluationReceipt
VideoExportPack
```
