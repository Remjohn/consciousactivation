# Template Preview / Template Atlas V1

## Definition

Template Preview / Template Atlas V1 makes production templates visually inspectable before they are approved or used.

It supports:

```text
SuperVisual templates
Carousel templates
Format 02 scene templates
```

The UI later should show:

```text
visual thumbnail
slot labels
sample data
JSON inspector
approve / duplicate / edit controls
```

V1 emits deterministic SVG preview read models. It does not call providers, Remotion, FFmpeg, or real render workers.
