# Conscious Interview Brief Skill

## 1. Purpose

The Conscious Interview Brief Skill compiles the first monthly production artifact for CMF Studio: the Interview Brief / Interview Asset Contract pack.

It is not a generic question generator. It reverse-engineers the old CCF trigger-first content pipeline and then passes it through CCP V9/V9.1 interview doctrine. The old CCF chain was:

```text
SCRE / CRAL signal discovery
-> primary signal packet
-> edge selection / pressure framing
-> coach provocation
-> authenticated response capture
-> primitive candidate generation
-> coalition survival and weighting
-> route recommendation
-> source artifact emission
```

In CMF Studio, the same intelligence is not reduced to a one-hop mapping from signal to question. It becomes a richer interview intelligence pathway:

```text
Guest Truth
+ Interviewer Resonance
+ Audience Reality / Context Premise
-> High-Identification Question Field
-> Research Field / Broad Primary Signal
-> Matrix of Edging pressure selection
-> target expression state
-> First-Line Anchor set
-> Depth Anchor
-> Interview Asset Contract
-> Complete Expression Session
-> landing evaluation
-> Expression Moment extraction
-> archetype / derivative / CMF route
```

The key shift is that CCF no longer compiles scripts first. It composes the intelligence needed for an Interview Asset Contract: a human-facing capture object designed to induce a specific expression state while preserving authentic speech, clean clipping starts, depth, routeability, and evaluable source truth.

## 2. Required Inputs

The skill cannot run from a topic, content format, or blank prompt. Required saturation context includes:

- CRAL / SCRE findings with source discipline;
- audience conversations, comments, objections, search questions, and social debates;
- Guest Dossier;
- Audience Reality Brief;
- Context Premise;
- Audience Deep Trigger Map;
- Interviewer Resonance Context;
- Matrix of Edging brief;
- primitive candidate packets;
- invariant, coalition, and edge-product references when migrated;
- Brand Context Version when available;
- prior evaluation receipts;
- failure corpus and hard negatives from generic/persona/topic-first attempts;
- CCF orchestration refs proving the old signal, edge, authentication, primitive, coalition, and route lineage was preserved.

## 3. Context Premise Role

Context Premise is the audience-side working hypothesis that explains why a question should matter now. It is not a persona, ICP paragraph, or content angle.

It must preserve:

- audience conversation evidence;
- recurring comments and objections;
- hermeneutical gaps;
- moral-emotional vectors;
- coping trajectory;
- promotion/prevention or regulatory focus when supported;
- audience/guest or audience/coach match logic;
- confidence, limitation, expiry, and risk if wrong.

Its output is useful only when it changes the question implications. If it does not change what we ask, it is not a production Context Premise.

## 4. Output Shape

The skill emits Interview Asset Contract candidates, not plain questions and not final content:

- main question;
- source pressure refs;
- Context Premise ref;
- Matrix brief ref;
- Interviewer Resonance ref;
- primitive candidate refs;
- invariant and coalition refs when present;
- target expression states;
- target archetype;
- asset derivatives;
- CMF routes;
- Guest Asset Pack potential;
- three First-Line Anchor options: cinematic, emotional, and reels hook;
- Depth Anchor;
- expected source material;
- clip start rule;
- depth eval rule;
- landing eval targets;
- repair followups for too historical, too abstract, too flat, and not clip-ready answers;
- intended guest reaction;
- intended extraction outcome;
- anti-centroid check;
- evaluation target hints.

## 5. Anti-Patterns

- Do not start from "what should we post about?"
- Do not generate questions from content formats.
- Do not collapse Context Premise into a target persona.
- Do not ignore audience comments and conversations.
- Do not ask polished biographical questions when the Matrix points to a pressure site.
- Do not treat primitives as style labels.
- Do not let a question enter the Interview Brief if it is not a full Interview Asset Contract.
- Do not skip First-Line Anchors, Depth Anchors, landing evaluation, repair followups, or route targets.

## 6. Quality Gate

A Conscious Interview Brief question passes only when it can answer:

1. Which CRAL signal or audience conversation made this question necessary?
2. Which Context Premise does it operationalize?
3. Which Matrix pressure or edge product does it activate?
4. Which primitive/invariant/coalition obligation does it preserve?
5. Which expression state is it trying to induce?
6. Which First-Line Anchor creates the clean start?
7. Which Depth Anchor prevents flattening?
8. Which landing types would count as success?
9. Which archetype, derivatives, and CMF routes could this expression support?
10. What repair followups are available if the answer becomes historical, abstract, flat, or not clip-ready?
11. What would make this question generic, coercive, or unsupported?
