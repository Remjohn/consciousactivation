# **Usage Protocol**

For each question:

1. Ask the **Main Question** naturally.  
2. Choose one of the 3 **First-Line Anchors** based on Claude’s current state.  
3. Say:

    *“Je te donne une première phrase, et tu la complètes à ta manière.”*

4. Use the **Depth Anchor** if the answer becomes too abstract, too historical, or too general.  
5. Use the **3 Narrative Instrumental Follow-ups** only if needed.
