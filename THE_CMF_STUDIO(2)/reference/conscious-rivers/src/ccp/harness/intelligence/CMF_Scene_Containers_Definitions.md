# CMF Scene Containers Definitions

This document is the canonical definition layer for CMF scene containers. A container is a fixed biological arc position, not a scene style.

## HOOK

- Purpose: win the first 3 seconds through a clear focal signal and immediate stakes.
- Neural basis: orienting response, fusiform face recognition, ventral attention activation.
- Output demand: single focal point, compressed ASL, low initial CLS, rapid meaning declaration.
- Default component: `HOOK`

## SETUP

- Purpose: create self-referential context so the viewer feels located inside the story.
- Neural basis: Default Mode Network and narrative self-projection.
- Output demand: warmer affect, slower rhythm, low information churn, psychological safety.
- Default component: `SETUP`

## CHALLENGE

- Purpose: establish stakes, opposition, or the cost of staying the same.
- Neural basis: amygdala activation, conflict framing, binary emotional contrast.
- Output demand: moderate CLS, rising tension, clear problem naming, sharper contrast than SETUP.
- Default component: `CHALLENGE`

## TURNING_POINT

- Purpose: deliver the felt revelation or peak reframe.
- Neural basis: peak snapshot formation, VAN plus frontal update, multi-network integration.
- Output demand: sensory peak, elevated CLS, premium emphasis, then a controlled drop toward stillness.
- Default component: `TURNING_POINT`

## RESOLUTION

- Purpose: convert the peak into closure, safety, and memory consolidation.
- Neural basis: hippocampal encoding, deceleration, closure recognition.
- Output demand: warm return, slower rhythm, low CLS, clear end-state stabilization.
- Default component: `RESOLUTION`

## VISION

- Purpose: fuse future identity with the call to action in the remembered end-state.
- Neural basis: Peak-End completion, ISC synchrony, prospective memory binding.
- Output demand: aspirational recognition, communal or future-state imagery, CTA overlap, warm premium finish.
- Default component: `THE_VISION`
