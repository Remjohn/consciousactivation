# CS-019 - Directional Semantics

CS-019 exists because movement direction carries narrative meaning. In CMF, motion should not be selected only for visual energy if its direction quietly says the wrong thing.

The subsystem reads motion vector, narrative function, cultural reading assumptions, and scene role. It succeeds when direction reinforces the intended meaning. It fails when the motion path encodes subtle contradiction.

CS-019 is especially useful in automated effect selection where technically valid motion can still be narratively wrong.

## Current CMF Thresholds

- empowerment defaults: upward or left-to-right
- challenge defaults: downward or right-to-left
- contradiction tolerance should remain minimal

## Good Example

An upward push supports a success reveal and makes the motion feel naturally aligned with the line.

## Failure Example

A descending tilt undercuts a triumphant beat and makes the scene feel subtly conflicted.
