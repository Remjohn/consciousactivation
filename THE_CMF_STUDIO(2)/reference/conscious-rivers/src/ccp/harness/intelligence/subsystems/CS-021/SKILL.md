# CS-021 Skill

## Purpose

Apply the Pan Speed Limiter so lateral motion stays legible and comfortable instead of collapsing into judder.

## Use This Skill When

- selecting pan or reveal motions
- validating horizontal traverses
- deciding whether a cut would serve the beat better than a rushed move

## Required Inputs

- pan duration
- frame width
- frame rate
- movement amplitude

## Decision Procedure

1. Measure the effective traverse speed.
2. Compare it against the safe viewing floor.
3. If the move is too fast, slow it down or replace it with a cut.

## Runtime Guidance

- Motion comfort is part of meaning delivery.
- A rushed pan makes the move itself salient.
- If a cut is cleaner, use the cut.
