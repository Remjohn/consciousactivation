# CS-030 - Neural Phase Router

CS-030 exists because different arc positions correspond to different attentional and mnemonic jobs. In CMF, scene parameters should therefore be routed from container-specific neural targets rather than from generic style habits.

The subsystem reads container type, target neural phase, emotional function, and the current scene parameters. It succeeds when motion, pacing, color, and saliency settings support the intended cognitive state. It fails when the scene is formally built like the wrong kind of beat.

CS-030 is the architecture bridge between neuroscience-informed containers and actual composition defaults.

## Current CMF Thresholds

- parameters must inherit from container-specific neural targets
- minimum neural-phase consistency score: 0.8

## Good Example

A vulnerability beat lowers motion intensity and raises intimacy because that is the correct neural-phase target.

## Failure Example

A scene marked vulnerability is built like a hype trailer, so the formal system contradicts the intended psychological job.
