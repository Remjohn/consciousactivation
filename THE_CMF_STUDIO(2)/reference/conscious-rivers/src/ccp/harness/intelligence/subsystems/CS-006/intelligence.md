# CS-006 - Gaze Continuity Checker

CS-006 exists because the viewer experiences a cut as smooth when attention survives the transition and as disruptive when the focal target jumps too far. In CMF, cut quality is therefore measured by perceptual economy, not by stylistic novelty.

The subsystem reads outgoing and incoming focal coordinates, cut type, motion cues, and scale change. It succeeds when the post-cut landing point stays close enough to preserve attentional momentum. It fails when the viewer must perform unnecessary reorientation.

CS-006 is especially useful in A-roll/B-roll alternation where otherwise valid inserts can still create friction if the focal point relocates too aggressively.

## Current CMF Thresholds

- preferred focal shift: within 15% of frame width
- larger jumps require explicit reorientation cues
- continuity should dominate over decorative transition style

## Good Example

The incoming reaction shot places the coach eyes near the same screen region as the previous proof focal point.

## Failure Example

The outgoing point is center-left and the incoming point is low-right, so the cut becomes the main event rather than the message.
