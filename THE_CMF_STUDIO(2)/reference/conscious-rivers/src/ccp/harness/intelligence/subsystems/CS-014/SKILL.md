# CS-014 Skill

## Purpose

Apply the Emotional Beat Limiter so short-form sequences stay emotionally legible instead of fragmenting into too many affective turns.

## Use This Skill When

- designing the emotional arc of a video
- reviewing scenes with multiple mood reversals
- compressing complex emotional plans into mobile-viable form

## Required Inputs

- per-scene emotional tags
- valence and arousal transitions
- beat count
- runtime

## Decision Procedure

1. Count distinct emotional beats.
2. Measure whether each turn has enough space to consolidate.
3. Collapse over-segmented arcs into a primary and secondary vector.

## Runtime Guidance

- Rich is not the same as crowded.
- One strong emotional line beats many weak shifts.
- If every scene introduces a new feeling identity, simplify.
