# CS-034 SKILL — Event Schema Activation Monitor

Use this subsystem when:
- Evaluating whether the narrative schema is sufficiently established before a key violation or surprise beat
- Diagnosing incoherence caused by too many event boundaries before the viewer has oriented
- Checking boundary density across the assembled manifest

Key outputs: `schema_activation_score`, `boundary_density_verdict`, `segmentation_advice`

Research: Zacks Event Segmentation Theory, Narrative Schema Theory (Mandler & Johnson)
