# CS-013 Skill

## Purpose

Apply the Variation Engine so CMF avoids technically correct repetition across consecutive videos.

## Use This Skill When

- selecting new scene vehicles for recurring formats
- auditing campaign repetition
- preventing identical timing and component patterns across recent outputs

## Required Inputs

- recent video history
- component usage history
- timing signatures
- motion/effect history
- tribe response signals

## Decision Procedure

1. Compare the current plan against recent output patterns.
2. Measure novelty without violating biological container functions.
3. Recommend substitutions at component, template, or parameter level.

## Runtime Guidance

- Change the vehicle before changing the biology.
- Novelty must preserve the arc.
- Repetition is often felt before it is named.
