# CS-010 Skill

## Purpose

Apply the Cross-Modal Audio Primer so the emotional field is established by sound before the image has to explain itself.

## Use This Skill When

- designing emotional transitions
- choosing whether a J-cut is required
- checking whether score timing supports the scene function

## Required Inputs

- audio key and mode
- timbral register
- beat map
- scene intent
- transition timing

## Decision Procedure

1. Determine the emotional function of the incoming beat.
2. Decide whether sound should lead the image.
3. Reject transitions where audio arrives too late or contradicts the target feeling.

## Runtime Guidance

- Audio is not garnish.
- Use J-cuts by default for emotionally sensitive transitions.
- If the image must do all the emotional work alone, the primer failed.
