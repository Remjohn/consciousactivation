# CS-012 - CTA Fusion Timer

CS-012 exists because fast-scroll environments erase intention quickly. If the emotional end-state and the CTA are separated by too much time, the behavioral impulse loses context before it can be acted on. In CMF, the ask must inherit the emotional momentum of the ending.

The subsystem reads end-state timing, CTA timing, gap duration, and continuity across the final window. It succeeds when the CTA lives inside the same emotional-action interval as the ending. It fails when decorative outro spacing, black fades, or detached tags separate the action prompt from the remembered end-state.

CS-012 belongs in the recall tier because it links memory to behavior. A strong ending without an action bridge is wasted momentum.

## Current CMF Thresholds

- max CTA gap from end-state: 3.0 seconds
- preferred fusion window: same 2 to 3 second end-state interval
- detached outro: not allowed by default

## Good Example

The final resolved image carries the CTA immediately while the emotion is still active.

## Failure Example

The story ends, then a separate branded tail appears before the CTA arrives.
