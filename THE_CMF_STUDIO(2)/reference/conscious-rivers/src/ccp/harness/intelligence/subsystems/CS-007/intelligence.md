# CS-007 - Theta Reset Validator

CS-007 exists because cuts trigger cognitive reset events that can either advance encoding or waste it. In CMF, each cut should therefore justify the reset it consumes by adding semantic, emotional, or perspectival value.

The subsystem reads cut timing, information delta, semantic novelty, and shot purpose. It succeeds when a cut introduces real progress. It fails when the sequence uses motion and reframing to simulate energy without actually moving meaning forward.

CS-007 is especially valuable in regeneration and polish passes where editors are tempted to add pace after the fact.

## Current CMF Thresholds

- every cut must introduce new information, angle, or emotional function
- redundant cut runs should be minimized
- reset value floor: 1 meaningful delta per cut

## Good Example

The cut introduces a new proof object and re-engages attention with a clear semantic shift.

## Failure Example

Three punch-ins happen during one sentence, but none of them introduce new information or emotional progress.
