# CS-031 Skill

## Purpose

Apply the Scene Type Selector so each biological container is filled by the best-fitting cinematic vehicle available, not by habit or convenience.

## Use This Skill When

- selecting a component/template pair for a container
- resolving conflicts between novelty, retention, and asset availability
- converting subsystem constraints into an actual scene choice

## Required Inputs

- container contracts
- component library
- asset inventory
- variation history
- subsystem constraints

## Decision Procedure

1. Filter candidate vehicles by biological fit.
2. Remove candidates that violate active subsystem constraints.
3. Rank the remainder by novelty, then preference.

## Runtime Guidance

- Biological fit comes first.
- Novelty matters only after validity is secured.
- This is the hinge between structure and creative inventory.
