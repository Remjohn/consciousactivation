# CS-020 Skill

## Purpose

Apply the Parallax Scaler so depth cues feel persuasive on mobile instead of synthetic or game-like.

## Use This Skill When

- exporting mobile-first motion designs
- adapting desktop parallax values to 9:16 outputs
- checking whether layered depth is too aggressive for the target screen

## Required Inputs

- target platform
- depth settings
- motion profile
- scene type

## Decision Procedure

1. Determine the display context.
2. Scale parallax depth to the platform's believable range.
3. Reject depth values that make the effect itself the subject.

## Runtime Guidance

- Mobile depth must be restrained.
- Quiet enhancement beats spectacular artificiality.
- The best parallax often disappears into felt realism.
