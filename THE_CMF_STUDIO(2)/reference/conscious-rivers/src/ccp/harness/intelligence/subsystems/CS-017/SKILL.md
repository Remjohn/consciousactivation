# CS-017 Skill

## Purpose

Apply the Temperature Arc so color temperature progresses with the emotional arc instead of staying flat or contradicting scene function.

## Use This Skill When

- planning scene-by-scene grade progression
- auditing adjacent scene temperature relationships
- adapting a brand palette to emotional truth

## Required Inputs

- scene order
- prior scene temperature
- target Kelvin values
- arc role
- brand constraints

## Decision Procedure

1. Determine the thermal role of each scene in the arc.
2. Compare adjacent scene temperatures for meaningful progression.
3. Revise static or contradictory thermal sequences.

## Runtime Guidance

- Temperature is a longitudinal storytelling signal.
- Do not let brand dogma flatten emotional progression.
- Use temperature changes that are perceptible but not cartoonish.
