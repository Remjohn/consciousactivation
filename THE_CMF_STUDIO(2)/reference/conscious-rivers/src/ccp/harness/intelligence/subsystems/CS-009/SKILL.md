# CS-009 Skill

## Purpose

Apply Emotional Contamination so B-roll absorbs the emotional field of the scene instead of fighting it with overly literal or conflicting meaning.

## Use This Skill When

- selecting B-roll against emotional A-roll
- checking whether a neutral insert will carry the intended feeling
- deciding between literal and absorptive support footage

## Required Inputs

- preceding emotional vector
- B-roll neutrality score
- contradiction risk
- sequence length
- emotional intensity

## Decision Procedure

1. Measure whether the B-roll is emotionally permeable or semantically dominant.
2. Reject inserts that contradict the established affective field.
3. Prefer simple, compositionally strong footage that can receive inherited meaning.

## Runtime Guidance

- Do not over-demand literal B-roll.
- Neutral footage can outperform explicit footage when the emotional field is strong.
- If the B-roll starts telling its own story, it is too semantically loud.
