# CS-016 Skill

## Purpose

Apply the PAD Color Vector Mapper so color is selected by affective coordinates instead of vague taste language.

## Use This Skill When

- choosing grade archetypes
- mapping scene emotion into measurable color parameters
- checking whether brightness and saturation support the intended scene role

## Required Inputs

- brightness distribution
- saturation distribution
- hue family
- target emotion
- scene role

## Decision Procedure

1. Map the intended scene feeling into Pleasure, Arousal, and Dominance targets.
2. Compare current grade characteristics against that vector.
3. Recommend a grade archetype that matches function rather than editor preference alone.

## Runtime Guidance

- Warm and cinematic is not a sufficient spec.
- Use PAD as the bridge from art direction to runtime logic.
- Grade for emotional truth, not generic style.
