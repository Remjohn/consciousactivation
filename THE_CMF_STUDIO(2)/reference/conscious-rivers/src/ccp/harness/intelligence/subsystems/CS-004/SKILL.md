Apply the ISC Structure Enforcer so the composition keeps the fixed biological arc and does not drift into structurally weak assembly that lowers audience synchrony.

2. Confirm every beat still performs a real arc function.
3. Check that free-form additions have not weakened shared structural guidance.
4. Return a structural synchrony score and `PASS`, `REVISE`, or `BLOCK`.
