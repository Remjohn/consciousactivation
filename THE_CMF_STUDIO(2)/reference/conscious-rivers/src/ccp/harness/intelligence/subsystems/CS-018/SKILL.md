# CS-018 Skill

## Purpose

Apply Presence/Arousal so motion choices maximize immersion per unit of arousal instead of chasing agitation.

## Use This Skill When

- selecting motion effects
- deciding between drift, shake, push, or parallax
- auditing whether an effect is immersive or merely hectic

## Required Inputs

- effect ID
- predicted presence score
- predicted arousal score
- CLS cost
- scene role

## Decision Procedure

1. Compare presence against arousal for each motion option.
2. Reserve high-arousal motion for genuine peak moments.
3. Reject effects that raise agitation without increasing felt participation.

## Runtime Guidance

- Presence often comes from subtle embodied motion.
- Cheap arousal burns attention.
- Vulnerability and teaching scenes usually need high presence, low arousal.
