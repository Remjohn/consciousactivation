# CS-013 - Variation Engine

CS-013 exists to prevent habituation across a creator's body of work. In CMF, the arc can stay stable while the cinematic vehicle changes enough to preserve alertness and perceived freshness.

The subsystem reads recent campaign history, component reuse, timing signatures, motion palettes, and tribe response. It succeeds when the system preserves biological fit while varying execution. It fails when the output becomes a predictable house style that is technically sound but increasingly dead.

CS-013 is one of the clearest safeguards against the AI-slop failure mode because the audience often notices repetition at a feeling level before they can explain it.

## Current CMF Thresholds

- no identical beat timing pattern in consecutive videos
- no identical component pattern in consecutive videos
- minimum novelty score: 0.7
- prioritize vehicle variation over arc violation

## Good Example

The SETUP function shifts from confession to archetypal visual setup while preserving the same biological job.

## Failure Example

Three consecutive campaign videos reuse the same HOOK type, pacing contour, and temperature arc with only cosmetic wording changes.
