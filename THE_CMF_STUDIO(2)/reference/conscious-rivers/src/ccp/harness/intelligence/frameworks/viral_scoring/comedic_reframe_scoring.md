# The Viral Scoring System: The Comedy Omni-Arc

## Core Identity
**Arc Type:** The Comedy Omni-Arc (Sonic Arc #10)
**Goal:** Disarm, Reveal, and Critique through Humor.
**Modes:** Self-Roast, Truth Bomb, Character, Glitch.

---

## 1. Mode DIAGNOSIS (The Sorter)
First, identify the dominant Comedy Mode. Use the corresponding Specificity Ladder.

| Mode | Key Signal |
|------|------------|
| **CR-1: Self-Roast** | "I am the problem." (Vulnerability) |
| **CR-2: Truth Bomb** | "Have you noticed..." (Relatability) |
| **CR-3: Character** | "Be like X..." (Satire/Impression) |
| **CR-4: Glitch** | "My brain is..." (Surreal/Absurd) |

## 3. Setup Validation (The Trap)
**Rule:** The Setup MUST hide the Punchline.
- **Deception Score (0-10):** How serious/normal does the setup sound?
- **Threshold:** Must be ≥8/10. NO "Funny stories".

---

## 4. Viral Trinity Thresholds

| Cluster | Key Focus | Threshold | Score Source |
|---------|-----------|-----------|--------------|
| **SETUP** | The Deception | **8/10** | Deception Score |
| **TWIST/ACT** | The Mechanism | **10/10** | Surprise Score |
| **PUNCH** | The Payoff | **9/10** | Mode Specific Ladder |
| **TRUTH** | The Insight | **8/10** | Emotion (Truth) |

## 5. The Unfunny Valley (Anti-Patterns)
**Penalty:** IMMEDIATE DISQUALIFICATION if any are present.

| Anti-Pattern | Description | Penalty |
|--------------|-------------|---------|
| **The "Dad Joke"** | Puns/wordplay without truth. | **-100% (REJECT)** |
| **The "Mean Girl"** | Punching DOWN (at victims/beginners). | **-100% (REJECT)** |
| **The "Just Kidding"** | Apologizing ("Haha, I'm joking"). | **-100% (REJECT)** |
| **The "Inside Joke"** | Context only verifiable by <3 people. | **-50%** |
| **The "Fake Laugh"** | Laughing at own setup before the punch. | **-30%** |
