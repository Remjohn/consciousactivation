# CMF Delegation Fixtures

`format02/` contains ten canonical reference scenarios. Each scenario points
to registered example messages and carries an executable lifecycle sequence.
`conformance/` contains positive and negative compatibility inputs used by the
validator suite. `compatibility/constitutional/` proves lossless Activative
lineage migration and rejects Expression Moment loss, unsupported
wrong-reading enforcement, and missing evaluator evidence. Fixtures do not
represent production delivery traffic.
