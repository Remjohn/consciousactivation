# CMF Delegation Protocol Reference

Status: `EXPERIMENTAL_GATE_OVERRIDE`

This package is the local, transport-neutral Stage 5 reference interpreter for
the `1.1.0-rc.2` contract candidate. It was implemented under explicit user
direction while the Stage 4 cross-repository verdict remains `FAIL`.

It provides schema, integrity, authority, compatibility, lifecycle,
idempotency, replay, audit, outbox, and in-memory persistence behavior. It does
not provide production transports, service deployment, Content Harness
creativity, Visual Production Plans, Visual Asset Editor production internals,
ComfyUI, VLM, or image generation.

Admission requires a pinned compatibility profile proving behavioral support
for all constitutional domains. The reference engine preserves the full-demand
canonical hash and exact lineage identities; it does not interpret or generate
the protected meaning.

This package is not authorized for publication or production integration until
the external Stage 4 blockers and signed release requirements are closed.
