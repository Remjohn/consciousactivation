# CCP Activative Intelligence & Visual Narrative Skill

## Governing law

```text
AI compiles the activation field.
The human reacts.
The reaction becomes content.
```

## Required execution order

```text
Evidence
→ Identity DNA
→ Context Premise
→ Resonance
→ Matrix of Edging
→ Activative Intelligence Pack
```

Then choose one branch.

### Conversational

```text
Activative Call
→ Human Reaction
→ Reaction Receipt
→ Expression Moment
→ recompilation
→ continue / elevate / close / final content routing
```

### Visual

```text
Visual Semantic Pack
→ Semiotic MCDA
→ Visual Narrative Program
→ format harness
→ feature contracts
→ Composition Asset Pack
→ Visual Syntax and production
```

## Activative Intelligence Pack checklist

- source references and frozen versions;
- Coach or Guest Identity DNA;
- Context Premise;
- Resonance Map;
- Edge Pressure;
- activation directions;
- target roles;
- stance and identity urges;
- participation design;
- intended reaction;
- micro-commitment;
- wrong-reading locks;
- evaluation contract.

## Interview rules

- The Interview Brief is compiled from Activative Intelligence Packs, not from generic research alone.
- Pre-session archetype and format routes are hypotheses.
- The final transcript authorizes Expression Moments and final Content Type Contracts.
- The guest owns truth and landing.
- First-Line and Depth Anchors execute the semantic plan; they do not create it.

## Visual rules

- Recognition intent and carrier are distinct.
- Visual Semantics and Visual Narrative are distinct.
- High-leverage features require contracts.
- Generated or transformed visuals require non-empty wrong-reading locks.
- No-text assets must pass the delete-caption test.
- The VAE may choose production method but may not alter semantic authority.

## Hard rejection conditions

Reject when:

- the topic is used as the edge;
- Identity DNA or Context Premise is unsupported;
- roles or desired reaction are absent;
- a final generator is asked to carry the full upstream theory;
- a visual demand lacks recognition, narrative, feature, or wrong-reading context;
- an interview route is finalized before transcript evidence;
- a semantic failure is repaired through visual polish.

## Repair order

```text
source / authority
→ Identity DNA or Context Premise
→ edge and activation
→ conversational call or visual semantics
→ visual narrative / format sequence
→ feature contracts / composition
→ production binding
→ rendering polish
```
