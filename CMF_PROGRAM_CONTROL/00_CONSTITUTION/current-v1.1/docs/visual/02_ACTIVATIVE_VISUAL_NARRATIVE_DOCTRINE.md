# Doctrine — Activative Visual Narrative Intelligence

## Master definition

Activative Visual Narrative is the intelligence that sequences visual pattern matching and visual pattern interrupts so the viewer's attention moves through stimulation, captivation, prediction, payoff, affinity, and anticipation while entering an identity role.

It is not simply “telling a story with pictures.”

```text
Pattern Match gets the audience inside.
Pattern Interrupt stops and destabilizes expectation.
Prediction makes them participate.
Payoff reorganizes meaning.
Affinity makes the event human.
Anticipation keeps the story alive after the asset ends.
```

## Why this layer is necessary

Visual Semantics alone can select a meaningful object and still fail. The Slack-message poster was semantically relevant but narratively inert. The footprint loop represented repetition but did not captivate. The phone triptych created sequence but collapsed into the wrong story.

The missing questions were:

- What stops the eye?
- What familiar pattern is recognized?
- What violation or juxtaposition changes the pattern?
- What does the viewer predict?
- What cost or truth becomes irreversible?
- What human detail prevents sterility?
- What remains unresolved?
- What role does the viewer enter?

## Narrative is attention-state architecture

A narrative program can operate inside:

- one static frame;
- a triptych;
- a carousel;
- a flyer;
- a thumbnail;
- a short reel;
- a long storyboard.

A single frame can contain several states through eye path and layered inference. A carousel can spread them across slides. A reel can time them across cuts.

## Core output

The `VisualNarrativeProgram` declares:

- format harness;
- pattern-matching plan;
- pattern-interrupt plan;
- attention-state sequence;
- viewer role progression;
- juxtaposition and violation operators;
- prediction gaps;
- payoff;
- affinity field;
- anticipation residue;
- feature contracts;
- wrong-reading locks;
- BBOX intent;
- format-specific evals.

## Governing test

Do not ask:

> What image represents this idea?

Ask:

> What sequence of familiar cues and violated expectations makes the target perform a mental action?
