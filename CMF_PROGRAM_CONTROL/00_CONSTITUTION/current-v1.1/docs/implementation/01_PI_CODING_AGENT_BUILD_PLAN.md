## Build doctrine

The Activative Intelligence Constitution is the semantic source of truth. The Builder, Interview Expression Harness, Delegation Protocol, and Visual Asset Editor remain separately versioned repositories with explicit contract boundaries.

BMAD is optional workflow tooling. Coding agents may work directly from the PRDs, amendments, `AGENTS.md`, technical-specification templates, and readiness gates.
