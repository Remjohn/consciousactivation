# V2.1 Dependency-Safe Spec Writing Wave Plan

The recovery graph has 23 topological waves. Wave 0 is authorized by the specification-work receipt. A later wave opens only after every required upstream draft has a writing receipt, an eligible draft state, and an exact SHA-256. Specs within a wave have disjoint output paths and may run in parallel under one controller.
