This phase implements the deterministic AIR semantic production path governed by
`TS-AIR-003`, `TS-AIR-015`, and `TS-AIR-016`. Read the exact Specs, Phase 5 AIR status,
and `CMF_PROGRAM_CONTROL/03_PROGRAM_STATUS/PHASE_05_SEMANTIC_PRODUCTION_COMPILER/`

packages, reusable animation scene packages, and Activation Transfer contracts/evidence.
