## Runtime truth

### Executed locally

- FFmpeg and ffprobe source inspection and rendering;
- actual H.264/AAC source-led short artifact;
- actual PNG SuperVisual and Carousel slides;
- actual PDF Carousel package;
- actual non-Format-02 animation MP4;
- static artifact decode and dimension reparse;
- cut-boundary before/after frame evidence.

### Compiled but not executed

- Remotion runtime binding;
- HyperFrames runtime binding.

### Conditional runtime

The static renderer uses `skia-python` when available. In the validated environment it uses `SKIA_COMPATIBLE_REFERENCE_RASTERIZER`. This is useful development evidence but is not real Skia worker proof and is not production-authorized.

## PRETEXT boundary

The Phase 6 `PretextEngine` is a deterministic integer reference measurement and wrapping implementation. It proves the Pipeline measurement contract, text-fit gate, and no-rewrite behavior. It does not claim conformance with an unavailable external PRETEXT binary, exact font shaping, multilingual shaping, or final production font evidence.

## Evaluation boundary

The implementation validates hard deterministic gates and enforces producer/evaluator identity separation. It creates actual rendered evidence. It does not call a certified VLM evaluator, and semantic visual judgment remains explicitly pending where required.
