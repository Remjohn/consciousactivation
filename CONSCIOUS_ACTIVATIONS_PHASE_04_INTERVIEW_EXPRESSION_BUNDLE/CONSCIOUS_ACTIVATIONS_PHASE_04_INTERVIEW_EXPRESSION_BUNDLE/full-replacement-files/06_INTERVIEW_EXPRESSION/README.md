# Interview Expression — Phase 4 deterministic development runtime

This product admits imported and Brief-led interview sources, compiles exact transcript and visual evidence, governs tags, Anchor Hits, Expression Moments and Reaction Receipts, and emits a Canonical Interview Source Package plus Asset Package Spec for AIR and Pipeline consumption.

The implementation is development-only. It does not perform external transcription, autonomous semantic interpretation, media editing, VAE production, Format 02 activation, production publication or certification.

## Commands

- `python -m conscious_activations_interview_expression bootstrap --json`
- `python -m conscious_activations_interview_expression demo --json`
- `python -m conscious_activations_interview_expression export-schemas <dir> --json`
