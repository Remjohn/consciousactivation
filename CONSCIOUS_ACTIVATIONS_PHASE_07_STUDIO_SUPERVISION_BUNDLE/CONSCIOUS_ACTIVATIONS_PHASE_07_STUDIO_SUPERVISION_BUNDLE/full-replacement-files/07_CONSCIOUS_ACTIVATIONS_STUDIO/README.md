# Conscious Activations Studio

The Studio is the supervisory, correction, review and ship-authority surface for Atomic Harness production.

Phase 7 provides a deterministic development runtime for:

- campaign order and autonomy policy;
- Harness-to-Studio surface routing;
- Control Tower and category-native workbench projections;
- canonical timeline projection;
- natural-language and direct-manipulation revision compilation;
- selective rerun planning;
- HumanResolution and programming-material capture;
- explicit ship decisions;
- audit and replay export.

## Commands

```bash
npm run build
npm test
npm run health
npm run demo
```

The generated demo is a development fixture. It is not a production Studio deployment.
