# activative-production-spine 0.1.0-dev.1

A development-only shared contract snapshot for the first application-foundation phase.

## Scope

The snapshot defines the minimum cross-product envelope types needed to establish:

- immutable references;
- exact authority and actor identity;
- artifact and source-span references;
- typed failures;
- command, event, and receipt envelopes;
- product handoffs;
- execution fingerprints;
- product status projections.

It includes deterministic Python and TypeScript declarations, positive and negative
fixtures, and a clean-room validator.

## Claim boundary

This package is unsigned, non-production, and not certified. It does not ratify the
V2.1 candidate authority, activate Format 02, authorize VAE Stage 5, or implement
product semantics or execution workflows.
