# Constitutional Alignment Charter

This active repository contains substantial validated work under a previous PRD authority. It must not restart.

The current amended PRD is adopted through bounded alignment batches. Existing artifacts remain authoritative where the amendment does not change them.

## Artifact verdicts

- `UNCHANGED`
- `PATCH_REQUIRED`
- `NEW_ARTIFACT`
- `SUPERSEDED`
- `BLOCKED`

## Safety laws

1. Do not erase or broadly rewrite validated work.
2. Do not implement beyond the repository's current authorization state.
3. Do not resolve constitutional or cross-product contradictions silently.
4. Record human decisions explicitly.
5. Validate each batch before advancing.
