# Release 1 — Format 02 Reference Vertical Slice

## Why Format 02

Format 02 exercises the most demanding foundations needed for the Visual Asset Editor:

- recurring identities;
- registry-driven pose, expression, gesture and gaze;
- controlled composition and negative space;
- continuity and beneficial recurrence;
- ComfyUI/local-model generation;
- LoRA or identity adaptation;
- transparent cutouts and simple scenes;
- image-conditioned geometry;
- VLM semantic and composition evaluation;
- targeted repair;
- downstream Remotion consumption.

## Source laws

- maximum three active elements;
- guide character, text field, meaning object;
- 9:16;
- no lip sync;
- deterministic text;
- large negative space;
- prop attachment points;
- avoid centered front-facing gaze;
- Remotion-first runtime.

## Complete proof

A Release 1 run must:

1. receive a demand from a real Format 02 harness;
2. retrieve identity and syntax context;
3. compile a Visual Production Plan;
4. choose and bind certified ComfyUI capabilities;
5. route to local or cloud compute;
6. produce a bounded candidate portfolio;
7. evaluate asset and composition;
8. run at least one targeted repair case;
9. promote an immutable master and variants;
10. return geometry and result contracts;
11. place the asset in a real Remotion scene;
12. preserve events, lineage, budget and evaluation receipts.

## Non-claim

This slice does not certify lip sync, complex multi-character animation, long-form video, every asset family, or every provider.
