Phase 2 implements the deterministic semantic lifecycle core for Conscious Activations.
It compiles and persists governed semantic objects; it does not call external models,
manufacture human truth, render media, activate VAE Stage 5, or authorize production.

- deterministic Primitive and archetype evidence retrieval;
- Psychological Role Inside a Tension contracts;
- Primitive Bindings, Coalition Contracts, Coalition Signatures, and Edge Products;
- archetype coalition programs with current-validation requirements;
- Brand Context Version, Guest Voice DNA, Visual DNA, and RSCS receipts;

- strict runtime schema metadata and a deterministic synthetic demonstration.
