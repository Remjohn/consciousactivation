# Observability plan

Required events: `ST-04.01:CapabilityOwnershipCompiled`, `ST-04.01:CapabilityCoverageValidated`, `ST-04.01:OwnershipEvidenceValidated`, `ST-04.01:OutcomeVerified`, `ST-04.01:CompilationReplayReturned`, `ST-04.01:CapabilityOwnershipInvalidated`, and `ST-04.01:OutcomeRejected`.

Each observation includes run, Story, Harness IR, constitutional report, graph, receipt, authority, version, provenance, capability count, owner-kind counts, reliability/cost evidence coverage, correlation, causation, command, stream, outcome, invalidation, and typed failure context. Do not log source payloads, artifact bytes, personal data, secrets, prompts, or external telemetry.
