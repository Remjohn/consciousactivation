# Non-Goals

- Designing, registering, drafting, packaging, evaluating, promoting, deprecating, revoking, discovering, or executing a skill.
- Creating a Skill Design Brief when the formal result is no demonstrated gap.
- Creating canonical skills, Harness-local adaptations, adapter implementations, recipes, bindings, packages, or JIT capsules.
- Seeding a real-profile or production registry.
- Implementing ST-05.03 or any later Story.
- Implementing workflow execution, Atomic Harness Definition compilation, or compiled-Harness Development Capsule generation.
- Using Format 02, conversational, Interview Expression, ReelCast, VAE, Delegation runtime, providers, evaluators, network, GPU, ComfyUI, Control Tower, publication, or certification behavior.
- Changing governance, schemas, planning baselines, authorities, Program Control, VAE, or Delegation files during implementation.
- Claiming production Harnesses never require skills.
