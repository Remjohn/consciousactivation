# Security and Authority Boundaries

- Only the capsule-authorized code actor may compile the definition; human ratification evidence remains attributable and immutable.
- The compiler may reference but never rewrite Source Locks, boundaries, ratification, Harness IR, precedence evidence, modules, Phase Graphs, contexts, or skill decisions.
- Category constitutions, production certification, and external product behavior remain human/external-owner authority and are not inferred.
- VAE and Delegation contracts/runtimes are not dependencies in this mode. No credentials, network, subprocess, provider, model, GPU, database, or production transport is permitted.
- Canonical inputs are hash-verified before use. Ambiguity, drift, stale authority, borrowed decisions, or missing evidence fails closed.
- Observations contain identifiers and hashes only; no secret, machine path, protected example, or external payload is emitted.
