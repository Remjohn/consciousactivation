# Builder V1.2 Constitutional Contracts

These files define documentation-time contract schemas and fixtures for the approved Builder V1.2 alignment. They are not production implementation, do not execute interviews, and do not implement the Visual Asset Editor or Delegation Protocol.

The authoritative contract catalog is `CONTRACT_REGISTRY.yaml`. JSON Schemas live in `schemas/`; deliberately non-production fixtures live in `examples/`. Run `python docs/contracts/tools/validate_contracts.py` before advancing from Batch B.

The runtime law is **Activation First**. **Visual Syntax First** remains the harness-development evidence order. Sparse downstream tokens are valid only while frozen rich-object lineage remains addressable.
