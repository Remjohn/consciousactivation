# Delegation V1.2 — Activative Object and Lineage Amendment

Delegation continues to own transport, lifecycle, compatibility, cancellation, amendment, supersession, and result delivery.

Representative messages must preserve:

- exact semantic object references and schemas;
- source package and source-time lineage;
- Final Script, Primitive coalition, archetype, role/tension, and Activation Transfer references;
- product-owner and consumer expectations;
- limitations, invalidation triggers, and supersession relationships;
- evaluation and handoff receipts.

Delegation does not interpret the semantic meaning of these objects and does not choose production strategy.
