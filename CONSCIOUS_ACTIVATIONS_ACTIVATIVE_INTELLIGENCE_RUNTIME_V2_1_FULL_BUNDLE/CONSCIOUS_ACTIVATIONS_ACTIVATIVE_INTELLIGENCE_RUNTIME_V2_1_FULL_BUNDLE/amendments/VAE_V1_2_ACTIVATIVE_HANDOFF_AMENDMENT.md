# Visual Asset Editor V1.2 — Activative Handoff Amendment

The VAE receives typed visual demands whose upstream semantic authority is already resolved.

New or deepened demand fields include:

- source package, Expression Moment, Final Script, and source-time lineage references;
- psychological role and tension;
- Primitive Coalition, Coalition Signature, Edge Product, and Primitive misuse locks;
- archetype coalition and category function;
- Visual DNA and real-life reference package;
- Activation Transfer invariants;
- composition role, BBOX function, negative-space function, edge-integrity constraints, and wrong-reading locks;
- animation-scene package and identity reference when the asset supports reusable 2D animation.

GNM remains a VAE-owned geometric capability. It may produce bounded facial geometry, expression, pose, gaze, meshes, and reference passes. It does not define Identity DNA, emotional truth, Expression Moments, Primitive coalitions, archetypes, or Final Scripts.
