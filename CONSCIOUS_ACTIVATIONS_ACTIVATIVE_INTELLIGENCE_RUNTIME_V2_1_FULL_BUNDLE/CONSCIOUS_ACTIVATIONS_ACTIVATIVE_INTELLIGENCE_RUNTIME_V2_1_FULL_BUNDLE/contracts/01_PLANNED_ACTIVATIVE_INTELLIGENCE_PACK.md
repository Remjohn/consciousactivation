# Planned Activative Intelligence Pack Contract

## Purpose

Freeze pre-event activation intent without representing it as observed truth.

## Required semantic fields

- pack ID and version;
- domain;
- epistemic state = planned;
- source premise reference;
- Identity DNA, Context Premise, Resonance, and Matrix of Edging refs;
- relationship and interviewer context refs where applicable;
- candidate portfolio;
- selected hypothesis;
- target state;
- pressure dose;
- participation design;
- intended reaction;
- smallest useful commitment;
- counteractivation hypotheses;
- wrong-reading locks;
- evaluation contract.

## Invariants

- no Reaction Receipt is required;
- no Expression Moment is implied;
- selected hypothesis remains linked to rejected alternatives;
- sparse generation tokens preserve stable refs to the rich pack.
