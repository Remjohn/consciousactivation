# Interview Asset Contract

The atomic source-activation program.

Required semantic components:

- target Narrative State;
- current-state hypothesis;
- First-Line Anchor;
- Depth Anchor;
- main question;
- follow-up branches;
- expected material;
- anticipated signals;
- pressure ceiling;
- recovery options;
- landing criteria;
- derivative hypotheses;
- hard negatives and wrong readings.

A question string alone is not a valid contract.
