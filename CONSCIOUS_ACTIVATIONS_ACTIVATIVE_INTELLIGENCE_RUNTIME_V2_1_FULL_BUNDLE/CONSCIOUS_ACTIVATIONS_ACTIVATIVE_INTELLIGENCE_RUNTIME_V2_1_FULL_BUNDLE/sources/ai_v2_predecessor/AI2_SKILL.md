# Integrated Activative Intelligence V2

## Use this skill when

- compiling an Activative Intelligence lifecycle;
- designing an activative comment or interview;
- observing and resolving a human reaction;
- approving Expression Moments;
- transferring source activation into a derivative;
- building a campaign activation program;
- diagnosing activative drift;
- compiling a visual or conversational format harness;
- converting operator corrections into programming material.

## Required operating sequence

1. Select activation domain.
2. Lock source and epistemic states.
3. retrieve Identity DNA, Context Premise, Resonance, and Matrix of Edging references;
4. generate a diverse Activation Hypothesis Portfolio;
5. choose a bounded Source, Relationship, or Audience Activation Program;
6. declare the intended state transition, role, dose, and smallest useful commitment;
7. execute through the owning harness;
8. observe reaction or rendered output without inventing evidence;
9. issue a Reaction Receipt or Evaluation Receipt;
10. resolve Expression Moments or candidate assets;
11. compile Activation Transfer Contracts for derivatives;
12. record HumanResolutionEpisodes for meaningful human decisions;
13. promote learning only after repeated, scoped evidence.

## Hard gates

- planned and observed fields are separate;
- every key claim has source lineage;
- the atomic interview unit is an Interview Asset Contract;
- no derivative bypasses a Canonical Source Package or explicit imported-source declaration;
- no visual generator invents gaze, hands, posture, or wrong-reading locks when those features are material;
- every candidate search preserves rejected and repaired candidates;
- every repair identifies the responsible layer first;
- canonical Identity DNA changes only through a profile-resolution event.

## Output preference

Prefer rich typed objects upstream and sparse tokens downstream. The generator receives the minimum useful contract, while the system preserves stable references to the full authoritative objects.
