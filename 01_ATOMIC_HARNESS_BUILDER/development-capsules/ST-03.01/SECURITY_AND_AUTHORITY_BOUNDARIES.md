# Security and Authority Boundaries

Deterministic code validates and commits question selection. A bounded recommendation
is advisory. Only `HUMAN` may own final constitutional decisions; this Story records
no human answer, final value or signature. Unsupported paths or authority fail HG-001.
The Constitution and PRD are referenced, never redefined. Evidence and Draft Model are
read-only. External products, runtime, protected labels and certification remain out
of scope.
