# Acceptance Criteria

1. Given the three exact governing registries, when structural profiles compile, then
   the seven edited-video formats belong only to Short-Form Edited Video and Format 02
   belongs only to 2D Character Animation.
2. Given Format 02, when its structural contract is inspected, then every FR-144
   character-performance and continuity registry requirement is explicit.
3. Given the four conversational profile IDs, when the registry compiles, then they
   remain under the fifth category, structural, non-production, and uncertified.
4. Given a cross-category reuse, missing Format 02 registry requirement, altered input,
   false certification, or unauthorized actor, when compilation is attempted, then it
   fails closed with no partial state.
5. Given identical governed inputs or a repeated command, when compilation is replayed,
   then byte-identical output and the original receipt result; conflicting reuse fails.
6. Given any output, when inspected, then both constitutional laws, complete provenance,
   `contract_compatible`-only Format 02 state, and explicit empty-profile categories are preserved.
