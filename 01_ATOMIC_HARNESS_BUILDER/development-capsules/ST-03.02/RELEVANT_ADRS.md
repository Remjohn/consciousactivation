# Relevant ADRs

- `ADR-005` is normative: human, agent and code authority are distinct; code cannot
  substitute for human ratification.
- `ADR-003` governs atomic ledger/artifact behavior through the development adapter.
- `ADR-004` governs canonical serialization and immutable versioning.
- `ADR-002` reserves canonical Harness IR compilation to its own aggregate and Story;
  this Story emits a governed decision amendment only.
