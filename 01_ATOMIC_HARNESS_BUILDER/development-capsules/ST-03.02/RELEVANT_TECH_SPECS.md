# Relevant Technical Specifications

- `TS-05`: dependency-driven Genesis, one question, separate answer/decision,
  transactional amendment, cascade lock and resumable memory.
- `TS-01`: governed run, typed events, authority and resume behavior.
- `TS-04`: frozen atomic boundary and unratified Draft Harness Model input.
- `TS-07`, `TS-12`, `TS-13`, `TS-14`: later consumers, observability, security and
  workflow boundaries. They do not authorize runtime work here.
