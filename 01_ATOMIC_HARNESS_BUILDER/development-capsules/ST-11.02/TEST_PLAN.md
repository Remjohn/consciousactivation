# Test plan

- Acceptance: exact FR coverage, three vertical outcomes, first working increment,
  backward dependency order and complete evidence fields.
- Negative: horizontal layers, forward/cyclic edges, missing tests, broad scope,
  implementation authorization, production/certification and external behavior.
- Trust: parent capsule identity/hash/receipt, authority actor and immutable input pin.
- Lifecycle: idempotency, conflicting command, atomic failure, replay, upstream
  invalidation and historical retrieval.
- Determinism: canonical bytes and receipt equality across fresh contexts.
- Observability: attributable success, replay and rejection records.
- Architecture: exact source set, no external imports, databases, transports or APIs.
- Regression: full repository suite after Story tests.
