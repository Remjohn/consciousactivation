immutable, queryable evidence index from every descriptor in the active `SourceLock`
and preserve the distinction between observed data, governed status, knowledge
status and provenance.

The implementation is generic. `BD-004` activates only when a real reference-profile
corpus is selected; `HD-006` activates only when Human Reaction material is present.
Neither applies to the repository-owned non-personal synthetic source used by this
bounded implementation.

The standing human authority is `AUTHORIZE FULL ATOMIC HARNESS BUILDER IMPLEMENTATION
CAMPAIGN`. It authorizes implementation only after this capsule, its hashes, the
`ST-01.02` receipt and the BQA trust-correction receipt validate.

The capsule does not authorize saturation decisions, visual inference, Format 02,
conversational processing, VAE or Delegation behavior, external runtimes, production
persistence, publication or certification.
