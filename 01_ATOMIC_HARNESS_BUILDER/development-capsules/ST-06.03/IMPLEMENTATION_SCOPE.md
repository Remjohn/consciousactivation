# Implementation Scope

Add one pure domain compiler and one application command boundary that compile a
governed structural profile plus frozen Shared Activative Core into deterministic
`CategoryNativeSyntax` and `ActivativeSequenceProgram` artifacts and one immutable
receipt.

The compiler must implement these distinct structures:

- Short-Form Edited Video: time-state and edited-sequence grammar.
- 2D Character Animation: character-performance, staging, state, and continuity
  grammar; Format 02 must not alias edited-video grammar.
- Carousels: slide-role, swipe-order, and cross-slide continuity grammar.
- Supervisuals: single-frame hierarchy and reading order with temporal structure
  explicitly `NOT_APPLICABLE`.
- Conversational Activation / Human Expression: turn relationship, Activative Call,
  Reaction Receipt reference, Expression Moment reference, landing, close, and
  micro-commitment structures; never a timeline or generic document.

Generic non-Activative tasks retain an explicit, justified `NOT_APPLICABLE` result and
cannot be silently assigned category semantics. All output remains development-only,
non-production, and uncertified.
