# ADR-014: Format 02 Release 1 Reference

Status: `ACCEPTED`

Owners: Product lead and category steward. Trace: D007, D008, D009, D010, D022, D030, D031, D032; FR-143, FR-167, FR-168; TS-15. Empirical blockers: BD-004, BD-008.

## Context

Release 1 requires one complete Atomic Content Harness path. Current direction names Format 02 Minimal Coach Theatre. The PRD classifies it under 2D Character Animation because character performance, registry state, staging, and continuity are constitutive rather than cosmetic.

## Decision

Use Format 02 under the 2D Character Animation constitution as the Release 1 reference slice. It must exercise every Builder subsystem from source lock through downstream signed effectiveness evidence. Other categories and external targets remain structurally supported and `UNCERTIFIED`.

## Alternatives

- Short-form edited-video reference: not selected by current product direction.
- Multiple simultaneous references: rejected for Release 1 because it dilutes complete vertical proof.
- Format 02 treated as a theme of edited video: rejected by FR-143 and category authority.

## Interfaces, Data, And Errors

`ReferenceSlice`, Format 02 profile/registries, 2D category constitution, corpus manifest, workflow profile, benchmark plan, Development Capsule, and downstream result receipt. Missing corpus, thresholds, human authority, or downstream identity blocks certification.

## Authority, Security, And Determinism

Product/category humans ratify policy; evidence/benchmark stewards control corpora; Builder deterministically compiles; external Content Harness owner implements. Protected labels and external credentials remain isolated.

## Consequences

Positive: concrete full-spine proof with category-native visual and temporal intelligence. Cost: corpus/registry/benchmark investment and no general claim from one slice.

## Observability, Performance, Migration

Trace every requirement, node, artifact, cost, decision, gate, repair, and downstream outcome to one slice identity. Changes create a new slice version. General certification remains deferred.

## Verification

End-to-end shadow, workflow, fault, security, benchmark, capsule, handoff, and downstream receipt tests must pass without certifying other profiles.
