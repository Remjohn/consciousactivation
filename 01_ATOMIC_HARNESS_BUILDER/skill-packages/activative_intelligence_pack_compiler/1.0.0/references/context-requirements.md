# Context requirements

The minimum complete context is:

- immutable source and authority references;
- Identity DNA reference;
- Audience Context Premise reference;
- Live Premise evidence references;
- Resonance Map reference;
- Matrix of Edging reference;
- explicit edge pressure and format goal;
- desired roles and Activative Call constraints;
- desired reaction and micro-commitment;
- non-empty wrong-reading locks;
- conversational and visual applicability decisions with reasons.

Unrelated history, hidden conversation context, external provider configuration,
credentials, production assets, and reaction-derived evidence are forbidden context.
